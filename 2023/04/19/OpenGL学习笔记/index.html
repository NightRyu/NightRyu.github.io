图形渲染管线

管线渲染流程

描述一下什么是管线渲染

这里是一个比较简单的管线渲染流程。

  1. 顶点着色器(Vertex Shader)将各个顶点作为输入，将一种3D坐标转换为另一种3D坐标。

  2. 图元装配(Primitive Assembly)将顶点着色器的顶点作为输入，将输入的数据矩阵变换位置，装配成图元形状。

  3. 将图元装配的结果输入到几何着色器(Geometry Shader)。通过输入的图元形式的一系列顶点的集合，通过新产生的顶点构造出新的图元来生成其他形状

  4. 几何着色器输出后进入光栅化阶段(Rasterization Stage)，将图元映射成屏幕上的像素，生成片段着色器使用的片段(Fregment)

  5. 片段着色器(Fragment Shader，又称为片元着色器)计算每个像素的最终颜色

着色器语言

着色器之间通常不能通信，之间所有的沟通仅通过输入和输出

通常使用名为GLSL(OpenGL Shading Language)的类C语言编写

在开头处声明版本，输入输出变量、uniform，随后是函数入口点 void mian()。在main函数中处理所有输入变量，并将结果送入输出变量

常用的模板

    #version 330 core

    in vector_type in_variable_name;
    out vector_typr out_variable_name;

    uniform type uniform_name;

    void main(){
        //处理输入的数据并进行图形操作
        ...
        //将输出的数据送入输出变量
        out_variable_name = 处理的数据;
    }

GLSL 中的向量是一个可以包含有1、2、3 或者 4 个分量的容器，分量的类型可以是前面默认基础类型的任意一个。它们可以是下面的形式(n代表分量的数量)

类型 含义
vec+n 包含n个float分量的向量
bvec+n 包含n个boolean分量的向量
ivec+n 包含n个int分量的向量
uvec+n 包含n个unsigned int分量的向量
dvec+n 包含n个double分量的向量

想要获取分量，可以使用 向量名.x 的格式获取第一个分量，同理使用 .y .z .w获取第2、3、4个分量

可以通过组合xyzw的形式进行向量重组

    //如果向量构造器中只提供了一个标量参数，则向量中所有值都会设定为该参数，即
    //color={1.0,1.0,1.0,1.0}
    vec4 color = vec4(1.0);

    //向量重组
    vec2 someVec;
    vec4 differentVec = someVec.xyxx;
    vec3 anotherVec = differentVec.zyw;
    vec4 otherVec = someVec.xxxx + anotherVec.yxzy;
